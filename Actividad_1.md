## Convierte el número decimal 22 a binario.
22 / 2 = 11 → residuo 0
11 / 2 = 5 → residuo 1
5 / 2 = 2 → residuo 1
2 / 2 = 1 → residuo 0
1 / 2 = 0 → residuo 1
